### [markdown语法](http://wowubuntu.com/markdown/#link "快速索引")

* 区块元素
>- 段落和换行， 至少一个空行为换行；
>- 标题， \#\~\#\#\#\#\#\#， 分别对应h1～h6
>- 区块引用，加上>，嵌套块区使用>>
>- 列表，\*开头无须， 数字开头有序
>- 代码区块，\`\`\`开头， \`\`\`结尾
>- 分隔线，\*\*\*

* 区段元素
>- 链接，行内：\[百度\]\(https:www.baidu.com "百度搜索"\)，参考：\[百度\]\[bdId\]
>- 强调，\*或者\_包裹
>- 代码，\`a\`
>- 图片，!\[Alt text\]\(/path/to/img.jpg "Optional title"\)。 或者使用image标签
